# Influences-of-sputtered-crater-geometry-on-glow-discharge-spec-trometry-depth-profile
Datas and codes for an academic paper titled "Influences of sputtered crater geometry on glow discharge spec-trometry depth profile"
file ‘CRAS_conb’ is use for create convolution results as only consider the CRAS model, and file 'MRI_CRA_conv1' is that consider both MRI and CRAS modles.
file 'crater' is use for create crater shape as setting of CRAS parameters and layer structure.
